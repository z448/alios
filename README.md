NAME

alios - tool for quick jumps into app folders


INSTALLATION

```bash
git clone https://github.com/z448/alios
cd alios
```
switch to 'root' and install with 'dpkg'

```bash
dpkg -i alios.deb
```

switch to 'mobile' and add to your '~/.bashrc' file

```bash
echo "alios -p && . ~/.alios" >> ~/.bashrc
```
